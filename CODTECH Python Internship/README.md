# CODTECH Python Internship Tasks

This repository contains all four mandatory Python internship tasks.

## Tasks
1. API Integration & Data Visualization
2. Automated PDF Report Generation
3. AI Chatbot with NLP
4. Machine Learning Model

## How to run
Create a virtual environment, install dependencies, and run each task from its folder.

Technologies:
- Python
- requests, matplotlib
- pandas, reportlab
- nltk
- scikit-learn, jupyter
